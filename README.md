# Dev Lake

## Quick start

### Required
- [Install Golang](https://golang.org/doc/install)

### Developer Setup
```shell
# clone lake repository
$ git clone https://github.com/merico-dev/lake.git
# enter lake directory
$ cd lake
# get packages
$ go get
# build 
$ go build
# copy .env.example to .env
$ cp .env.example .env
$ ./lake
```
